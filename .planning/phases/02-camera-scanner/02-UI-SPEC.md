## Mandatory Developer Guidelines (Skills Usage)

> L'usage des compétences (skills) Flutter locales installées est **STRICTEMENT OBLIGATOIRE** pour l'agent de développement exécuteur afin de garantir la production de code qualitatif, de respecter l'architecture MVVM et d'assurer une couverture de tests adéquate.

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (e.g. between icon and text in SnackBar) |
| sm | 8px | Compact element spacing, border radius of visual buttons, scan box border corner radius |
| md | 16px | Default element spacing, padding around control buttons, dialog padding |
| lg | 24px | Section padding, margin around camera preview bounds |
| xl | 32px | Layout gaps, spacing below error screen icon, spacing around description text |
| 2xl | 48px | Major section breaks, sizing of floating action buttons |
| 3xl | 64px | Page-level spacing, height of scanner guide corners |

Exceptions:
- Scan guide box: responsive size equal to 70% of the screen width, constrained to a minimum of `200dp` and a maximum of `320dp` (Decision D-07).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFFFFF` | Background of the permission error screen, application surfaces (Light Theme) |
| Secondary (30%) | `#E1E2EC` | SnackBar background container, custom overlay paint shadow opacity (0.5 black overlay) |
| Accent (10%) | `#87CEEB` | Active scan window border corners (Sky Blue), active floating torch toggle button (when enabled) |
| Destructive | `#BA1A1A` | Permission denied error icon, open settings CTA button text or background |

Accent reserved for:
- Torch toggle button background color when active/enabled.
- Borders and angles of the responsive visual scan guide frame.
- Primary action buttons (e.g., settings redirection button on permission error screen).

## Visual Hierarchy & Focal Points

- **Primary Screen (Scanner):**
  - **Focal Point:** The responsive visual scan guide box (centered on the screen) is the primary visual anchor.
  - **Visual Hierarchy:** Visual scan guide box -> Floating torch toggle button (top-right) -> SnackBar notification / temporary results.
- **Permission Error Screen:**
  - **Focal Point:** The primary action button `Ouvrir les paramètres`.
  - **Visual Hierarchy:** Destructive error icon -> Description text -> `Ouvrir les paramètres` CTA button.
