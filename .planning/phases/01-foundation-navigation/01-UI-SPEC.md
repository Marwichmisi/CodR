## Spacing Scale

Declared values (logical pixels â Flutter `EdgeInsets` / `SizedBox`):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, tight element spacing |
| sm | 8px | Compact element spacing, internal card padding |
| md | 16px | Default element spacing, body content padding |
| lg | 24px | Section padding, AppBar title area |
| xl | 32px | Layout gaps, placeholder icon-to-title spacing |
| 2xl | 48px | Major section breaks, bottom nav safe area |
| 3xl | 64px | Page-level spacing, large vertical gaps |

Exceptions: none. All spacing must be multiples of 4.

**Usage rules:**
- Placeholder screens: 32px gap between icon and title, 16px gap between title and subtitle.
- AppBar padding: 16px horizontal (default Material).
- Bottom nav: standard Material 3 `NavigationBar` spacing (no overrides).
- Scaffold body content: 16px horizontal padding.

- Placeholder screen subtitle/body: Body size (16px, w400).
- Bottom nav tab labels: Label size (14px, w400).
- Never use sizes outside this set. No hardcoded font values â always use `Theme.of(context).textTheme`.

## Color

Material 3 `ColorScheme.fromSeed()` with seed color **`Color(0xFF87CEEB)`** (sky blue). The scheme is derived algorithmically â no manual overrides beyond the seed.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `scheme.surface` (~#FFFFFF light) | Page backgrounds, scaffold body |
| Secondary (30%) | `scheme.surfaceContainerHighest` (~#E8E8E8 light) | Bottom nav bar, cards, sidebar surfaces |
| Accent (10%) | `scheme.primary` (~#00796B teal-ish from seed) | Active tab indicator, AppBar icons, interactive highlights |
| Destructive | `scheme.error` (#B3261E) | Destructive actions only (future phases) |

**Accent reserved for (explicit list):**
- Active `NavigationBar` destination indicator and icon color
- `AppBar` foreground icons (when applicable)
- FAB or primary action button (future phases)
- Nothing else â text, backgrounds, and surfaces must use dominant/secondary only

**Source:** D-01 (seed = bleu ciel + vert), D-02 (light mode only), D-12 (active tab = seed color).

**Light mode only.** No dark mode configuration. `ThemeData` uses `brightness: Brightness.light`.
