## Component Inventory

### HistoryScreen (main deliverable)

**Structure:**
```
Scaffold
├── AppBar(title: "Historique")
└── Body: LayoutBuilder → ConstrainedBox(maxWidth: 480)
    └── Column
        ├── SearchBar (TextField) — sticky top
        ├── TypeFilter (Row of FilterChips) — sticky below search
        └── Expanded(ListView.builder) — scrollable list
```

**SearchBar:**
- Widget: `TextField` with `InputDecoration`
- Hint text: `"Rechercher dans l'historique..."`
- Prefix icon: `Icons.search`
- Clear button: `Icons.clear` (visible when text non-empty)
- Background: `#F4FAFC` (secondary surface)
- Border: `OutlineInputBorder` with `borderRadius: 12`
- Padding: `EdgeInsets.symmetric(horizontal: 16, vertical: 12)`
- Behavior: Real-time filtering on `onChanged` — no submit button

**TypeFilter:**
- Widget: `Row` of 3 `FilterChip` widgets
- Chips: `"Tout"`, `"Scans"`, `"Générations"`
- Default selected: `"Tout"`
- Selected chip: accent background (`#87CEEB` at 20% opacity), accent border
- Unselected chip: `#F4FAFC` background, grey border
- Spacing: `gap: 8` between chips
- Behavior: Single selection (toggle), combined with search filter

**List Items (RecordCard):**
- Widget: `Card` wrapping `ListTile`
- Card elevation: `0` (flat, Material 3 style)
- Card color: `#FFFFFF`
- Card margin: `EdgeInsets.symmetric(horizontal: 16, vertical: 4)`
- Card shape: `RoundedRectangleBorder(borderRadius: 12)`
- ListTile leading: Type icon — `Icons.qr_code_scanner` (scan) or `Icons.qr_code` (generation), tinted with accent color
- ListTile title: Content preview — `maxLines: 1`, `overflow: ellipsis`, body 16px regular
- ListTile subtitle: Relative timestamp — label 14px regular, color `onSurfaceVariant`
- ListTile trailing: None (timestamp is in subtitle)
- ListTile contentPadding: `EdgeInsets.symmetric(horizontal: 16, vertical: 4)`

**Swipe-to-Delete:**
- Widget: `Dismissible` wrapping each `Card`
- Key: `ValueKey(record.id)`
- Direction: `DismissDirection.endToStart` (swipe left only)
- Background: `#D32F2F` (destructive red) with trash icon aligned right
- ConfirmDismiss: Shows `AlertDialog` before allowing dismiss
- Behavior: On confirm → delete from DB, refresh list. On cancel → record stays.

**AlertDialog (Delete Confirmation):**
- Title: `"Supprimer l'entrée"`
- Content: `"Voulez-vous vraiment supprimer cette entrée de l'historique ? Cette action est irréversible."`
- Icon: `Icons.delete_forever` tinted `#D32F2F`, size 48, above title
- Actions: `"Annuler"` (text button, no action) + `"Supprimer"` (text button, `#D32F2F` color)
- Barrier dismissible: `true` (tap outside closes)

**Empty State:**
- Icon: `Icons.history` size 64, color `onSurfaceVariant`
- Heading: `"Aucun historique"` (display 24px semibold)
- Body: `"Vos scans et générations apparaîtront ici"` (body 16px regular, `onSurfaceVariant`)
- Layout: Centered Column with `mainAxisAlignment: center`
- Spacing: icon → 32px → heading → 8px → body

**Error State:**
- Icon: `Icons.error_outline` size 64, color `#D32F2F`
- Heading: `"Erreur de chargement"` (heading 20px semibold)
- Body: `"Impossible de charger l'historique. Réessayez plus tard."` (body 16px regular)
- No retry button (data is local, unlikely to fail transiently)

## Interaction Contracts

### Search
- **Trigger:** Text input in search bar
- **Behavior:** Real-time substring match on `record.content`
- **Case sensitivity:** Case-insensitive
- **Empty input:** Shows all records (no filter)
- **No results:** Shows empty state with different copy: `"Aucun résultat"` / `"Aucune entrée ne correspond à votre recherche"`

### Filter
- **Trigger:** Tap on FilterChip
- **Behavior:** Single selection, combined with search
- **States:** All (default) → Scans only → Générations only
- **Visual:** Selected chip gets accent background, unselected reverts to secondary

### Swipe-to-Delete
- **Gesture:** Swipe left on list item
- **Visual feedback:** Red background revealed with trash icon
- **Confirmation:** AlertDialog appears on release
- **Confirm:** Record deleted, list refreshes, no SnackBar (per D-06)
- **Cancel:** Record slides back, no change
- **Edge case:** Swipe during animation — Dismissible handles natively

### List Loading
- **Initial load:** Show `CircularProgressIndicator` centered while ViewModel loads from DB
- **Empty after load:** Show empty state
- **Records present:** Show ListView immediately

## Timestamp Formatting

| Age | Format | Example |
|-----|--------|---------|
| < 1 min | `"à l'instant"` | `à l'instant` |
| 1–59 min | `"il y a X min"` | `il y a 12 min` |
| 1–23 h | `"il y a X h"` | `il y a 3 h` |
| 24–47 h | `"hier"` | `hier` |
| ≥ 48 h | `"le JJ/MM/AAAA"` | `le 27/06/2026` |

Threshold: 48 hours switches from relative to absolute date.
