### Responsive Behavior
- Wrap body in `LayoutBuilder` + `ConstrainedBox(maxWidth: 480)` (existing pattern from `GeneratorScreen`)
- On screens wider than 600dp, constrain content to 480dp centered
- QR preview scales proportionally within constrained width

### Element Specifications

#### TextField
- `maxLines: 4`, `minLines: 2`
- `maxLength: 250` (built-in counter)
- Counter format: `n/250`
- Counter text color: `colorScheme.primary` normally, changes to `#D32F2F` (destructive) at 250/250
- `decoration: InputDecoration(hintText: "Entrez du texte ou une URL...")`

#### URL Badge
- Shown only when URL is detected (text contains `.` and no spaces, or starts with `http://`, `https://`, `www.`)
- Style: `Container` with `RoundedRectangleBorder(borderRadius: 8px)`
- Background: `colorScheme.primary` (sky blue) at 10% opacity
- Border: 1px solid `colorScheme.primary`
- Text: "URL dГ©tectГ©e" in `Label` size (14px), weight 400
- Icon: `Icons.link` (16px) before text
- Position: directly below TextField, with `SizedBox(height: 8)` spacing

#### QR Preview / Placeholder
- **When text is empty**: Center-aligned column with `Icons.qr_code` (64px, `colorScheme.primary`) and text "Saisissez du texte pour gГ©nГ©rer un QR code" (`Body` size, `colorScheme.onSurfaceVariant`)
- **When text is present**: `QrImage` widget from `qr_flutter`, data = decoded text (with `https://` prefix if URL detected), size = 200px (responsive: min(200, maxWidth - 48)), eyeStyle/fill/color = black/white (default)
- **Debounce**: 300ms вҖ” QR updates 300ms after user stops typing

#### Action Buttons
- Row of 3 buttons below QR preview with `SizedBox(height: 24)` spacing above
- Each button: `FilledButton` (primary) or `OutlinedButton` (secondary)
- Layout: Row with `MainAxisAlignment.spaceEvenly`
- Button 1: "Sauvegarder" вҖ” `Icons.save_alt` вҖ” triggers gallery save
- Button 2: "Partager" вҖ” `Icons.share` вҖ” triggers native share
- Button 3: "Copier" вҖ” `Icons.content_copy` вҖ” copies text to clipboard
- Button width: flexible, equal distribution
- Button height: 48px (Material 3 default touch target)
- Button icon size: 20px
- Button label: 14px, weight 500

#### SnackBar Feedback
- Position: `SnackBarBehavior.floating`
- Duration: 2 seconds
- Messages:
  - Save success: "QR sauvegardГ© !" (green/background color)
  - Save error: "Erreur lors de la sauvegarde" (red/destructive)
  - Share success: no SnackBar (share sheet is self-evident)
  - Copy success: "CopiГ© !" (green/background color)
  - Permission denied: "Permission galerie refusГ©e. Activez-la dans les rГ©glages." (red/destructive)
