<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User input → TextField | Text crosses into ViewModel for QR encoding |
| ViewModel → SaverGallery | Image bytes cross platform boundary to native gallery |
| ViewModel → SharePlus | Image file crosses platform boundary to native share sheet |
| Permission request → OS | Permission status crosses platform boundary |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-04-04 | Information Disclosure | Gallery save | low | accept | Save is explicit user action; image saved to user's own gallery; no data leaves device |
| T-04-05 | Tampering | QR image capture | low | accept | RepaintBoundary capture is deterministic; no external input to the capture process |
| T-04-06 | Denial of Service | Share sheet | low | accept | share_plus handles platform share sheet; no custom code in share path |
| T-04-07 | Information Disclosure | Auto-save prohibition | high | mitigate | Save/share/copy only trigger on explicit button press; no auto-save on text input or screen load |
</threat_model>
