# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-06-29
**Phases:** 1-5
**Total Plans:** 14

## Overview

Application mobile Flutter qui scanne des QR codes via caméra et génère des QR codes à partir de texte/URLs. L'app affiche le contenu scanné avec des actions (ouvrir, copier, partager), gère l'historique avec recherche et filtres, et est 100% offline.

## Milestone Summary

**Key Decisions:**

- Flutter comme framework — projet d'apprentissage Flutter, choix imposé par le défi
- MVVM avec setState — suffisant pour 2-3 écrans, pas besoin de Riverpod/Bloc
- Material 3 — design standard Google, bien supporté par Flutter
- Pas de backend — app simple, données en local, pas de besoin de sync
- Texte + URLs uniquement — simplicité pour v1, les types QR complexes ajoutent de la complexité
- Skills Flutter obligatoires — suivre les bonnes pratiques via les skills installés

**Issues Resolved:**

- sqflite version incompatible avec Dart SDK 3.11.0 (utilisé ^2.3.0)
- WidgetsBindingObserver pour le cycle de vie caméra
- Anti-rebond de 2s pour éviter les détections multiples
- Overflow des filtres sur écrans étroits (Row → Wrap)
- Permissions galerie MVVM-compliant via ViewModel

**Technical Debt:**

- StorageService nullable dans les ViewModels (par design pour l'isolation des tests)
- Imports inutilisés dans navigation_test.dart (avertissements test uniquement)
